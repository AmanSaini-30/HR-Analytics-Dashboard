# HR Analytics Dashboard - Employee Attrition & Insights

## Project Goal
The objective of this project was to analyze HR data to identify key factors contributing to employee attrition and to provide actionable insights for improving employee retention and satisfaction.

## Key Features & Insights
* **Comprehensive Metrics:** Tracked Total Employees (1470), Attrition Rate (16.12%), and Average Monthly Pay ($6.5K).
* **Demographic Analysis:** Visualized employee distribution by age group and gender, identifying the 25-34 age bracket as the largest group.
* **Job Satisfaction:** Created a matrix analyzing satisfaction ratings across different job roles.
* **Attrition Drivers:** Identified that Sales Representatives have the highest attrition rate (nearly 40%).

## Technologies Used
* **Power BI:** For data visualization and dashboard creation.
* **DAX:** For calculating custom measures and attrition rates.
* **Excel:** For initial data cleaning and preparation.
